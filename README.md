# Chronicles of Elia

Chronicles of Elia is a personal full stack project focused on personalized English learning through a fantasy RPG-inspired mobile experience.

The app adapts learning paths according to the user's English level, profession, interests and real goals.

## Author
María Fernanda Rodríguez Trinidad

## Project Status

Currently in planning, documentation and initial development phase.

## Tech Stack

### Mobile
- React Native

- TypeScript

### Backend

- REST API
