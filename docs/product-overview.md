# Product Overview

## Project Name

Chronicles of Elia

## Description

Chronicles of Elia is a personal full stack project focused on personalized English learning through a fantasy RPG-inspired mobile experience.

The app adapts each learning path according to the user's English level, profession, interests and real goals.

## Problem

Most English learning apps offer general content for all users. However, people usually need English for specific real-life contexts.

For example, a software developer needs English for technical interviews, documentation, meetings and code reviews, while a chef needs English for recipes, ingredients, customer service and kitchen instructions.

Chronicles of Elia aims to make English learning more meaningful by connecting lessons with the learner's real professional and personal context.

## Value Proposition

Chronicles of Elia combines:

- Personalized English learning paths
- RPG-inspired experience
- Professional classes such as Dev, Chef and Artist
- Skill trees and missions
- AI-assisted practice and feedback
- Pedagogical structure based on real learning goals

## Pedagogical Foundation

Chronicles of Elia is based on a CEFR-inspired progression model.

Each English level defines the linguistic difficulty, while each professional path defines the context, vocabulary, examples and final communicative tasks.

The app does not treat one mission as mastery of a level. Instead, each mission represents a small communicative objective within a long-term learning path.

For internal tracking, each CEFR-inspired level is treated as a long-term journey with an estimated target of 500 study hours.

```text
CEFR-inspired level = language difficulty
Professional path = learning context
Mission = communicative objective
Progress = accumulated study, practice and performance
```

This allows the app to teach real English structures while adapting examples and tasks to the user's professional context.

For example, an A2 learner can practice present simple and routines through different professional paths:

```text
DEV:
I usually write documentation.

CHEF:
I usually prepare the ingredients.

ARTIST:
I usually sketch ideas.
```

The grammar level remains appropriate to A2, but the context changes according to the learner's profile.

## Visual Concept

The app is inspired by a fantasy forest adventure. Elia is the guide who accompanies the player through missions, lessons and challenges.

The user is treated as an adventurer. Their profession becomes their class, their learning goals become quests, and their English skills become abilities to unlock.

## Target Users

Initial users include young adults and adults who want to learn English for professional or personal goals, especially:

- Software developers
- Chefs
- Artists
- Travelers
- Customer service workers
- Career changers
